comment l'utiliser : 

*******************    backend      *****************************

créer un fichier .env à la racine et y ajouter: 
- le port : PORT
- le token d'authentification : JWT_SECRET
- l'url de la BD : MONGO_URI

puis executer la commande npm install

les fakes data se trouvent dans le dossier DB data


******************      frontend        ****************************
